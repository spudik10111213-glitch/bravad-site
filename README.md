# BRAVAD

Учебный сайт студии 3D/CG-дизайна для контрольной работы по дисциплине WebDesign.

## Страницы

- `index.html` - главная страница, сверстана с использованием Flexbox.
- `projects.html` - страница проектов, сетка карточек выполнена через CSS Grid.
- `contacts.html` - страница контактов и заявки, используется Bootstrap CDN.

## Технологии

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Bootstrap 5
- JavaScript

## JavaScript

На странице `contacts.html` форма заявки обрабатывается локальным скриптом `js/script.js`.
Скрипт проверяет основные поля, показывает Bootstrap Modal “Заявка отправлена” и очищает форму после отправки.
